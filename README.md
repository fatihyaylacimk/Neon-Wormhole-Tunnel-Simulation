# Neon Wormhole Tunnel Simulation

Neon Wormhole Tunnel Simulation is a fullscreen visual animation project created with HTML, CSS, and JavaScript.

This project is not a dashboard or a standard web application. It is a pure visual simulation focused on motion, depth, neon wireframe effects, and interactive creative coding.

---

## Project Purpose

The purpose of this project is to create a futuristic wormhole tunnel animation using only frontend technologies.

The simulation creates a moving tunnel effect with neon wireframe rings, star particles, glowing depth motion, and mouse-reactive bending. It is designed for GitHub portfolio, LinkedIn sharing, YouTube Shorts, and Instagram/Reels content.

---

## Main Features

- Fullscreen visual simulation
- Neon wormhole tunnel effect
- Moving wireframe rings
- Star particle flow
- Glowing center core
- Depth and speed illusion
- Mouse-reactive tunnel bending
- Smooth animation loop
- Dark futuristic background
- Pure HTML, CSS, and JavaScript

---

## Technologies Used

- HTML
- CSS
- JavaScript
- Canvas API
- `requestAnimationFrame`
- Custom perspective projection
- Creative coding animation logic
