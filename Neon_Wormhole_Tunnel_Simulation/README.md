# Neon Wormhole Tunnel Simulation

Neon Wormhole Tunnel Simulation is a fullscreen visual animation created with HTML, CSS and JavaScript.

It creates a futuristic tunnel/wormhole effect with neon wireframe rings, depth motion, star particles and mouse-reactive bending.

## Project Purpose

The purpose of this project is to create a code-based animated visual simulation suitable for GitHub, LinkedIn, YouTube Shorts and Instagram/Reels content.

This is not a dashboard or a standard web application.  
It is a pure visual simulation focused on motion, depth and creative coding.

## Main Features

- Fullscreen neon tunnel animation
- Wormhole-style depth motion
- Moving wireframe rings
- Star particle flow
- Mouse-reactive tunnel bending
- Glowing center core
- Smooth animation loop
- Pure HTML, CSS and JavaScript

## Technologies Used

- HTML
- CSS
- JavaScript
- Canvas API
- requestAnimationFrame
- Custom perspective projection

1. Download or clone the project.
2. Open the project folder.
3. Double-click `index.html`.
4. The visual simulation starts automatically.
